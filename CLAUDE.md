# World Cup Country Draft Project

Welcome to the **World Cup Country Draft** project!

This folder contains the complete conceptual specs, user flows, database schemas, and developer prompts to build the mini-fantasy web application for the 2026 FIFA World Cup.

## Folder Contents
* **[prd.md](file:///Users/radwan/Documents/Projects%20and%20Workflows/World%20Cup%20Game/prd.md):** The finalized product requirements including the validated **Moderate Multiplier model** (T1=1.0x, T2=1.25x, T3=1.6x, T4=2.0x), group lock constraints, and scoring equations.
* **[spec.md](file:///Users/radwan/Documents/Projects%20and%20Workflows/World%20Cup%20Game/spec.md):** The detailed technical architecture, system states (capsules draw, draft grid, live scoreboard), database schemas, and CSS/HTML design parameters.
* **[prompt.md](file:///Users/radwan/Documents/Projects%20and%20Workflows/World%20Cup%20Game/prompt.md):** The master developer builder prompt. When you are ready to construct this project, feed this prompt into a fresh coding session to generate the server-side code (`Code.gs`) and client dashboard (`Index.html`).
* **[ideas.md](file:///Users/radwan/Documents/Projects%20and%20Workflows/World%20Cup%20Game/ideas.md):** Brainstorming logs of alternative tournament game concepts.

## Coding Style & Instructions
* **Backend:** Server-side scripts should be coded in Google Apps Script syntax (`Code.gs`) communicating with a Google Sheet database.
* **Frontend:** Responsive client-side templates in standard HTML5, clean CSS transitions, and Vanilla JS (`Index.html`).
* **Aesthetic:** Dark mode dashboard with glassmorphism layouts, circular flags, and neon green/gold sport accents.
